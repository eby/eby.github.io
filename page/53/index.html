Expect Scripting

Linux.com has a small article on expect scripting. For those not familiar, expect scripts allow you to script interactive interfaces. In libraries, for example, you can use it to log in to the ILS and do some tasks at a set schedule. This can help make up for missing features or the lack of a cron facility. Bloglines Feed Access Control Namespace

Bloglines has released a namespace for specifying access control for your feed, presuming you provide RSS 2.0 or ATOM. This also presumes the software you use allows customization of feeds. Not sure if this will catch on with other aggregators. Libraries should still be careful about what content goes into feeds, but this offers a mechanism to help prevent accidental sharing of even non-identifiable data.
