Search Data and Feed Security

Two things that are worth looking at and thinking about. First is news that AOL released a data set of users' searches on purpose. The data was partially anonymized but since some of the searches can be identifying in themselves it’s possible that this might turn into a privacy fiasco for them. There also seems to be reports that there’s sensitive data such as social security in it. If you do search analysis or other data mining, think long and hard how to anonymize and protect data....
